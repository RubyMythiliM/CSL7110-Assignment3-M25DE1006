## Project Overview

This project implements different recommendation system techniques using the MovieLens Small Dataset. The objective is to explore and compare multiple recommender system approaches.

The implementation was completed using Python in Google Colab.

---

## Implemented Methods

1. Content-Based Filtering
   - TF-IDF based similarity using movie genres
   - User profile based recommendation

2. Collaborative Filtering
   - User-Based Collaborative Filtering
   - Item-Based Collaborative Filtering

3. Matrix Factorization
   - Singular Value Decomposition (SVD)

4. Hybrid Recommendation System
   - Combination of content-based and collaborative signals

5. Model Evaluation
   - RMSE (Root Mean Squared Error)
